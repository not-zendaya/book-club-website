# Book Club Website Recreation

This project is a recreation of a BookClubs.com website. The goal of the project was to practice front-end web development skills by reproducing the layout, design, and structure of the existing website page.

The recreated page includes:

- A hero section with a call-to-action
- Features section (meetups, voting, tracking reads, chat)
- Book of the Month section
- Recommendations section
- Testimonials section
